# 实验三：自动写诗（唐诗生成）

## 实验任务背景

自然语言生成是深度学习的重要应用方向之一。唐诗作为中国古典文学的精华，具有严格的格律要求和丰富的意象表达，是检验语言模型生成能力的理想任务。本实验基于 LSTM 循环神经网络，训练一个字符级的唐诗生成模型，实现给定首句后的自动续写。

## 实验目标与要求

1. 理解和掌握 LSTM 循环神经网络在文本生成中的应用
2. 掌握 Embedding、LSTM、LayerNorm 等 PyTorch 模块的使用
3. 基于唐诗数据集训练字符级语言模型，实现给定首句自动续写诗句
4. 网络结构设计需有自己的方案（与实验指导书示例不同）
5. 输出的诗句应尽可能满足汉语语法和表达习惯

## 数据集信息

- **来源**：课程提供的预处理唐诗数据集（tang.npz）
- **规模**：57,580 首唐诗
- **词汇量**：8,293 个汉字（含特殊符号）
- **序列格式**：每首诗固定 125 字符，右对齐，左侧用 `</s>` 填充
- **特殊符号**：
  - `<START>`（idx=8291）：句首标记
  - `<EOP>`（idx=8290）：句末标记
  - `</s>`（idx=8292）：填充符号
- **数据划分**：训练 80% / 验证 10% / 测试 10%

## 模型架构要点

- **Embedding**：将汉字索引映射为 384 维稠密向量
- **LSTM**：3 层单向 LSTM（hidden_dim=768），层间 Dropout=0.3
- **LayerNorm**：归一化 LSTM 输出，稳定深层网络训练
- **Dropout**：输出层前增加 Dropout=0.3
- **Linear**：投影到 8,293 维词表空间
- **总参数量**：~22.6M

与指导书示例的差异：3 层 LSTM（vs 单层）、引入 LayerNorm + Dropout 正则化、更大嵌入和隐层维度。

## 评价指标

- **困惑度（Perplexity）**：语言模型的标准评价指标，PPL = exp(cross_entropy_loss)。越低越好。
- **生成质量**：给定经典唐诗首句，人工评估续写诗句的语法正确性和语义连贯性。

# 实验一 手写数字识别实验报告

## 一、实验目的

1. 掌握卷积神经网络的基本原理。
2. 掌握 PyTorch 的基本用法以及卷积网络的构建方法。
3. 熟悉在 GPU 上进行模型训练与评估的基本流程。

## 二、实验环境

- 操作系统：Linux
- Python：3.13
- 深度学习框架：PyTorch 2.11.0+cu130、torchvision 0.26.0+cu130
- 训练硬件：NVIDIA GeForce RTX 5070
- 数据集：MNIST

## 三、实验原理

本实验采用卷积神经网络对 MNIST 手写数字进行分类。模型主体流程如下：

`Conv2d -> BatchNorm2d -> ReLU -> Conv2d -> BatchNorm2d -> ReLU -> MaxPool2d -> Dropout -> Conv2d -> BatchNorm2d -> ReLU -> Conv2d -> BatchNorm2d -> ReLU -> MaxPool2d -> Dropout -> Flatten -> Linear -> ReLU -> Dropout -> Linear`

该结构满足实验指导书中“卷积、激活、池化、展平、全连接输出”的组织方式。

## 四、实验步骤

1. 下载并加载 MNIST 数据集。
2. 对输入图像做张量化和标准化处理，并在训练集上加入轻量随机仿射增强。
3. 构建 CNN 分类模型。
4. 使用交叉熵损失函数和 Adam 优化器进行训练。
5. 划分验证集，保存验证准确率最高的模型。
6. 在测试集上评估模型最终准确率。

## 五、关键实现说明

- 数据预处理：`Normalize(mean=0.1307, std=0.3081)`
- 数据增强：`RandomAffine`
- 损失函数：`CrossEntropyLoss`
- 优化器：`Adam`
- 模型保存：保存验证集准确率最高的权重

## 七、结果分析

1. 卷积层能够有效提取图像局部特征，池化层降低了特征图尺寸并增强了模型鲁棒性。
2. 标准化和适度的数据增强有助于模型更快收敛并提升泛化性能。
3. 本次实验在测试集上达到了 99.51% 的准确率，超过指导书要求的 98%。
4. 采用 GPU 训练时，模型训练速度明显快于 CPU，能够在较短时间内完成多轮迭代。

## 八、结论

本实验完成了基于 PyTorch 的 MNIST 手写数字识别任务，实现了从数据加载、模型构建、训练到测试评估的完整流程。最终模型在测试集上取得了 99.51% 的准确率，满足并超过了实验指导书要求的 98% 指标。

# 实验一 手写数字识别实验报告

## 一、概述

- **任务**：图像分类——对手写数字灰度图像进行 0-9 十个类别的识别
- **数据集**：MNIST（28×28 单通道灰度图，训练集 60,000 张，测试集 10,000 张，共 10 类）
- **解决方案概要**：基于卷积神经网络（CNN）的监督分类，使用两层卷积-池化模块提取特征，后接全连接层进行分类输出

## 二、实验环境

- 操作系统：Linux
- Python：3.13
- 深度学习框架：PyTorch 2.11.0+cu130、torchvision 0.26.0+cu130
- 训练硬件：NVIDIA GeForce RTX 5070
- 数据集：MNIST

## 三、解决方案

### 3.1 网络结构设计

模型 `MNISTCNN` 采用经典的 CNN 结构，分为特征提取器 `features` 和分类器 `classifier` 两部分：

- **特征提取器**：两组卷积-BN-ReLU-MaxPool-Dropout 模块，每模块含两个卷积层。通道数从 1 → 32 → 64，逐步提取从边缘到语义的层级特征。MaxPool 将空间尺寸从 28×28 降采样至 7×7。
- **分类器**：Flatten → 全连接(3136→128) → ReLU → Dropout → 全连接(128→10)，输出 10 维 logits。

```python
class MNISTCNN(nn.Module):
    def __init__(self) -> None:
        super().__init__()
        self.features = nn.Sequential(
            nn.Conv2d(1, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(inplace=True),
            nn.Conv2d(32, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=2),
            nn.Dropout(p=0.25),
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(inplace=True),
            nn.Conv2d(64, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=2),
            nn.Dropout(p=0.25),
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(64 * 7 * 7, 128),
            nn.ReLU(inplace=True),
            nn.Dropout(p=0.5),
            nn.Linear(128, 10),
        )

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.classifier(self.features(x))
```

### 3.2 损失函数设计

采用 **交叉熵损失（CrossEntropyLoss）**，适用于多分类任务。PyTorch 内置实现自动融合了 softmax + NLLLoss，数值更稳定。

```python
criterion = nn.CrossEntropyLoss()
```

### 3.3 优化器设计

采用 **Adam 优化器**，学习率 0.001，权重衰减 1e-4。Adam 结合了动量与自适应学习率，收敛速度快且超参数不敏感，适合中小规模实验。训练过程中使用早停策略，保存验证准确率最高的模型。

```python
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3, weight_decay=1e-4)
```

### 3.4 创新点

- **数据增强**：训练集使用 `RandomAffine`（±8° 旋转、±10% 平移）增强，验证集和测试集仅做 `ToTensor` + `Normalize`，确保评估公平
- **AMP 混合精度训练**：使用 `torch.amp.autocast` + `GradScaler`，在保持精度的同时加速训练

2. 数据增强（RandomAffine）提升了模型对平移和旋转的鲁棒性，验证集准确率在后期仍保持上升趋势
3. Dropout 在两处使用（特征提取器 0.25、分类器 0.5），训练损失高于验证损失，但验证表现持续改善，有效防止过拟合
4. 第一层卷积核可视化为边缘检测器等基本模式，与图像的 GPU 纹理特征一致，说明网络从底层到高层构建了合理的特征层次

## 五、总结

本实验完成了基于 PyTorch 的 MNIST 手写数字识别任务，实现了从数据加载、模型构建、训练验证、到测试评估的完整流程。最终模型测试准确率 99.51%，超过指导书 98% 的要求。

可改进方向：
- 引入学习率调度策略（如 CosineAnnealingLR）进一步优化收敛
- 尝试更深层网络或残差连接以提升特征提取能力
- 使用更丰富的增强策略（如 RandomErasing 或 RandAugment）

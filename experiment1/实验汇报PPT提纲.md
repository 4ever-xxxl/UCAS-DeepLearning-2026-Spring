## 第 3 页：实验任务概述

- **任务定义**：对手写数字灰度图像进行 0-9 十个类别的分类
- **应用场景**：邮政编码识别、票据金额识别、银行表单数字化等
- **实验目标**：搭建 PyTorch CNN 模型，在 MNIST 测试集上达到 98% 以上准确率

## 第 4 页：数据集介绍

- **来源**：MNIST 手写数字数据库（LeCun et al., 1998）
- **规模**：训练集 60,000 张 → 训练 55K + 验证 5K，测试集 10,000 张
- **类别**：10 类（数字 0-9），各类别分布均衡
- **图像规格**：28×28 单通道灰度图，像素值归一化到 [0, 1]
- **预处理**：`Normalize(mean=0.1307, std=0.3081)`
- **数据增强**：`RandomAffine(±8° 旋转, ±10% 平移)`，仅用于训练集
- **样本展示**：如下图（各数字随机抽样）

## 第 5 页：模型结构设计

**整体架构：CNN（卷积神经网络）**

```
Input (1×28×28)
  → Conv2d(1→32, 3×3) → BN → ReLU
  → Conv2d(32→32, 3×3) → BN → ReLU → MaxPool → Dropout(0.25)
  → Conv2d(32→64, 3×3) → BN → ReLU
  → Conv2d(64→64, 3×3) → BN → ReLU → MaxPool → Dropout(0.25)
  → Flatten → Linear(3136→128) → ReLU → Dropout(0.5) → Linear(128→10)
```

- **特点**：两层卷积模块逐步提取特征（通道 1→32→64），MaxPool 降采样（28→14→7），Dropout 正则化防止过拟合
- **参数量**：约 44 万（轻量级）

## 第 6 页：损失函数与优化器

**损失函数**

- `CrossEntropyLoss`：多分类标准损失，内部融合 softmax + NLLLoss，数值稳定

**优化器**

- `Adam(lr=1e-3, weight_decay=1e-4)`：自适应学习率，收敛快，超参数鲁棒
- 无显式学习率调度，通过早停策略保存最优模型

**AMP 混合精度**

- 使用 `torch.amp.autocast` + `GradScaler` 加速，CUDA 可用时自动启用

## 第 7 页：创新点说明

- **数据增强策略**：`RandomAffine` 增加平移与旋转不变性，仅作用于训练集，验证/测试集保持纯净
- **AMP 混合精度训练**：在 RTX 5070 上利用 BF16 加速，训练效率提升约 1.5×，精度无损失
- **早停与模型保存**：每轮验证后记录最优模型，保证最终使用的是验证集最佳 checkpoint

1. **准确率超预期**：测试准确率 99.51%，显著超过 98% 要求，CNN 对 MNIST 分类任务效能充分验证

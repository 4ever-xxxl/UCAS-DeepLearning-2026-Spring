以下是该 5 分类任务的计算过程，结果均保留至小数点后 4 位：

### 1. 计算 Soft-Max 分类概率

**Soft-Max 公式：**


$$P(i) = \frac{e^{z_i}}{\sum_{j=1}^{5} e^{z_j}}$$

* **第一步：计算各 logits 的指数值 ($e^{z_i}$)**
* $e^{0.01} \approx 1.0101$
* $e^{-0.01} \approx 0.9900$
* $e^{-0.05} \approx 0.9512$
* $e^{0.02} \approx 1.0202$
* $e^{0.1} \approx 1.1052$


* **第二步：计算指数总和 ($\sum e^{z_j}$)**

$$\sum e^{z_j} = 1.0101 + 0.9900 + 0.9512 + 1.0202 + 1.1052 = 5.0767$$


* **第三步：计算各类别概率 ($P_i$)**
* $P_1 = 1.0101 / 5.0767 \approx \mathbf{0.1990}$
* $P_2 = 0.9900 / 5.0767 \approx \mathbf{0.1950}$
* $P_3 = 0.9512 / 5.0767 \approx \mathbf{0.1874}$
* $P_4 = 1.0202 / 5.0767 \approx \mathbf{0.2010}$
* $P_5 = 1.1052 / 5.0767 \approx \mathbf{0.2177}$



**Soft-Max 概率结果：** $[0.1990, 0.1950, 0.1874, 0.2010, 0.2177]$

---

### 2. 计算交叉熵损失 (Cross-Entropy Loss)

**交叉熵公式：**


$$L = -\sum_{i=1}^{5} y_i \ln(P_i)$$

由于标签为 One-hot 编码 $label = [0, 0, 0, 0, 1]$，只有第 5 个类别的乘积非零，公式简化为：


$$L = -\ln(P_5)$$

* **代入 $P_5$ 的精确计算值：**

$$L = -\ln(0.21769467...) \approx 1.5247$$



**交叉熵损失结果：** $\mathbf{1.5247}$

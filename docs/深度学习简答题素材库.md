## 一、名词解释（每题4分）

### 1. 集成学习

**核心定义**：通过组合多个基学习器（弱学习器）来构建一个更强模型的方法。

**关键点**：
- **Bagging**（如随机森林）：并行训练多个模型，投票/平均融合 → 降低方差
- **Boosting**（如AdaBoost、GBDT）：串行训练，每个模型纠正前一个的错误 → 降低偏差
- **Stacking**：用另一个模型来融合多个基模型的输出

**一句话**：多个弱模型组合成一个强模型，比单一模型更稳定、更准确。

---

### 2. 注意力机制

**核心定义**：让模型在处理输入时，动态地关注不同部分的重要程度，分配不同的权重。

**本质**：给定 Query，计算与每个 Key 的相似度作为权重，对 Value 进行加权求和。

**核心公式**：$\text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$

**关键点**：
- 解决了长序列的信息瓶颈问题（不再需要把所有信息压缩到一个固定向量中）
- Self-Attention：Q=K=V 来自同一序列，建模内部依赖
- Cross-Attention：Q来自一个序列，K/V来自另一个

**一句话**：让模型"看哪里"——给重要的部分更大的权重。

---

### 3. 自编码器

**核心定义**：一种无监督学习的神经网络，目标是将输入压缩成低维表示再重建出来。

**结构**：Encoder（编码）→ 瓶颈层（latent representation）→ Decoder（解码）

**关键点**：
- 本质是做**数据降维**（类似PCA的非线性版本）
- 变体：去噪自编码器（DAE）、变分自编码器（VAE）、稀疏自编码器
- VAE 在瓶颈层引入概率分布约束，可以用于生成新样本

**一句话**：把数据压缩再还原，学习到数据的紧凑表示。

---

### 4. 感受野

**核心定义**：卷积神经网络中，某层特征图上的一个像素点对应输入图像上的区域大小。

**关键点**：
- 浅层感受野小，关注局部纹理、边缘等细节
- 深层感受野大，关注全局语义信息
- 增大感受野的方式：增大卷积核、增加网络深度、使用空洞卷积（Dilated Convolution）
- 设计网络时需要考虑感受野是否覆盖了目标物体

**一句话**：每个神经元"看到"的输入区域有多大。

---

### 5. 残差连接

**核心定义**：让网络学习残差映射 $F(x) = H(x) - x$ 而非直接映射 $H(x)$，通过跳跃连接实现 $y = F(x) + x$。

**关键点**：
- 解决了深层网络的退化问题（degradation problem）
- 梯度可以直接通过跳跃连接回传到浅层 → 缓解梯度消失
- 使得训练上百层的网络成为可能
- 典型结构：ResNet 中的 Bottleneck Block

**一句话**：跳过一层或多层直接把输入加到输出上，让深层网络更容易训练。

## 二、简答题（每题10分）

### 6. 卷积神经网络的主要结构模块及功能

**答题框架：四大模块**

1. **卷积层（Convolutional Layer）**
   - 功能：用卷积核在输入上滑动提取局部特征
   - 特点：参数共享（同一个卷积核扫遍全图）+ 局部连接（每个神经元只连局部区域）
   - 输出：特征图（Feature Map）

2. **激活函数（Activation Function）**
   - 功能：引入非线性，使网络能学习复杂模式
   - 常用：ReLU（$f(x)=\max(0,x)$，计算快、缓解梯度消失）、Sigmoid、Tanh
   - 位置：每个卷积层/全连接层之后

3. **池化层（Pooling Layer）**
   - 功能：下采样，降低特征图空间尺寸
   - 作用：减少参数量、防止过拟合、提供平移不变性
   - 常用：最大池化（Max Pooling）、平均池化（Average Pooling）
   - 位置：卷积层之后

4. **全连接层（Fully Connected Layer）**
   - 功能：将学习到的高维特征映射到最终输出（类别概率等）
   - 位置：网络的最后几层
   - 注意：参数量最大，容易过拟合 → 常配合 Dropout 使用

> **加分点**：还可以额外提到批量归一化（Batch Normalization）——加速训练、缓解梯度问题。

---

### 7. 生成对抗网络的原理和训练过程

**核心思想**：两个网络博弈训练——生成器（Generator）学造假，判别器（Discriminator）学鉴假。

**结构**：
- **G**（生成器）：输入随机噪声 → 输出生成样本
- **D**（判别器）：输入样本 → 输出真/假概率

**训练过程（交替训练）**：
1. **固定 G**：从真实数据取 batch，从 G 取生成样本 → 训练 D 区分真假
   - 目标：$\max_D \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$
2. **固定 D**：用 G 生成样本 → 训练 G 让 D 误判为真
   - 目标：$\min_G \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$
3. 反复迭代直至达到纳什均衡（G 生成的样本 D 无法区分）

**关键问题**：
- 训练不稳定（模式坍塌、不收敛）
- 改进：DCGAN（用卷积）、WGAN（用 Wasserstein 距离替代 JS 散度）、WGAN-GP（梯度惩罚）

**一句话**：G 和 D 相互对抗、共同进步，像假币制造者和验钞员。

### 8. R-CNN 系列工作的改进思路

> 核心线索：从"慢"到"快"，把独立步骤逐步整合成一个端到端网络。

| 模型 | 关键改进 | 解决的问题 |
|------|----------|-----------|
| **R-CNN** | Selective Search 生成候选框 → 每个框缩放后分别送入 CNN 提取特征 → SVM 分类 + 回归 | 传统方法（滑动窗口+手工特征）速度慢、精度低 |
| **Fast R-CNN** | 整图过 CNN 得到特征图 → Selective Search 映射到特征图上 → RoI Pooling 统一尺寸 → 单网络同时分类+回归 | R-CNN 重复提特征（每个框单独过 CNN），速度极慢 |
| **Faster R-CNN** | **RPN（Region Proposal Network）**代替 Selective Search，共享特征提取网络 → 真正端到端 | Fast R-CNN 的候选框生成仍是瓶颈 |

**Faster R-CNN 的 RPN 细节**：
- 在特征图上滑动窗口，每个位置预设 k 个 anchor box
- 输出 2k 个分数（前景/背景）+ 4k 个坐标偏移
- 与检测网络共享卷积特征 → 近乎无代价的候选框生成

**一句话总结演进**：
> R-CNN 把候选框分别过 CNN → Fast R-CNN 共享特征图+RoI Pooling → Faster R-CNN 用 RPN 替代外部候选框算法，实现端到端。

### 9. 残差网络的理解，为什么能解决梯度消失

**核心理解（两个角度）**：

1. **从映射角度看**：网络学习残差 $F(x)=H(x)-x$ 比直接学习 $H(x)$ 更容易
   - 如果恒等映射是最优的，学习 $F(x)\rightarrow0$ 比学习 $H(x)\rightarrow x$ 简单
   - 实践中残差通常很小，网络只需要做微调

2. **从梯度流角度看（更重要）**：跳跃连接为梯度提供了"高速公路"
   - 令 $x_{l+1} = x_l + F(x_l, W_l)$
   - 则任意深层 $L$ 的输出：$x_L = x_l + \sum_{i=l}^{L-1} F(x_i, W_i)$
   - 反向传播时：$\frac{\partial Loss}{\partial x_l} = \frac{\partial Loss}{\partial x_L} \cdot (1 + \frac{\partial}{\partial x_l}\sum_{i=l}^{L-1} F(x_i, W_i))$
   - **关键**：梯度项中存在一个"1"，确保梯度不会消失，可以直接回传到任意浅层

**为什么解决了退化问题（degradation）**：
- 网络越深并非越好（实验发现 56 层比 20 层训练/测试误差都高）
- 原因不是过拟合，而是优化困难
- 残差连接让深层网络的优化变得简单，相当于"浅层网络 + 深层增量"

### 10. Transformer 网络 + Swin Transformer 的改进

#### Transformer 核心理解

**结构**：Encoder-Decoder 架构，全部基于 Attention，没有 RNN/CNN

**核心组件**：
1. **Self-Attention**：建模序列中任意两个位置的依赖关系
   - $\text{Attention}(Q,K,V) = \text{softmax}(QK^T/\sqrt{d_k})V$
   - 并行计算，解决了 RNN 无法并行的问题
2. **Multi-Head Attention**：多个注意力头捕获不同子空间的信息
3. **位置编码（Positional Encoding）**：弥补 Attention 缺乏位置感知的缺陷
4. **前馈网络（FFN）**：每个位置独立的两层全连接，引入非线性变换
5. **LayerNorm + 残差连接**：稳定训练

**为什么重要**：长程依赖建模能力强、可并行计算、可扩展到大规模数据和参数。

#### Swin Transformer 的改进

**核心问题**：标准 ViT（Vision Transformer）直接把图像切成 patch，存在的问题：
1. 计算复杂度是图像尺寸的平方（$O(N^2)$）
2. 缺少多尺度信息（只做固定尺度的 patch 划分）
3. 没有 CNN 的局部先验

**Swin Transformer 的改进**：

| 改进 | 做法 | 解决的问题 |
|------|------|-----------|
| **窗口自注意力（W-MSA）** | 将特征图划分成非重叠窗口，只在窗口内做 Self-Attention | 计算复杂度从 $O(N^2)$ 降至 $O(N \cdot M^2)$（M为窗口大小），大幅降低计算量 |
| **移动窗口（Shifted Window）** | 相邻层错位划分窗口，实现跨窗口信息交互 | W-MSA 的窗口间无信息流动，限制了感受野 |
| **层次化结构（Hierarchical）** | 通过 Patch Merging 逐步合并相邻 patch，形成多尺度特征金字塔 | 获得类似 CNN 的多尺度表示，适合密集预测任务（检测、分割） |
| **相对位置偏置** | 在 Attention 计算中加入相对位置编码 | 比绝对位置编码更适应不同分辨率的输入 |

**一句话**：Swin Transformer 用窗口注意力解决 ViT 的计算量问题，用移动窗口和层次化结构解决信息流动和多尺度问题。

## 三、计算题（10分）

### Valid 卷积手算要点

**审题注意**：
- Valid 卷积 = 不填充，只在输入完全覆盖卷积核时计算
- 步长为 1
- 激活函数是 ReLU（负数置 0）
- **卷积核不翻转**（即直接做逐元素相乘再求和，不是互相关，注意：深度学习框架中的"卷积"实际是互相关，题目强调不翻转说明就是做互相关运算）

**计算步骤**：
1. 确定输出尺寸：$(H - K + 1) \times (W - K + 1)$，其中 $K$ 是卷积核尺寸
2. 从左上角开始滑动，每个位置逐元素相乘再求和
3. 对结果应用 ReLU：负数 → 0，正数保持不变

> **考试技巧**：先写公式再代数字，分步计算避免出错。时间允许的话验算一遍。

## 四、设计题（20分）

### 设计题通用策略

设计题核心不是"默写现有模型"，而是展示**思考过程**。准备几个**万能模块**，考场上灵活组合：

**万能创新模块库**：
1. **注意力机制** → 加在瓶颈处或跳跃连接处
2. **多尺度融合** → 不同分辨率的特征做融合
3. **残差/密集连接** → 改善梯度流动
4. **知识蒸馏** → 用大模型指导小模型
5. **对比学习/自监督预训练** → 充分利用无标签数据
6. **频率域分析**（FFT/DCT）→ 在频率域做处理
7. **扩散过程** → 逐步去噪生成

**答题结构**：
1. **总体框架图**（几句话描述）
2. **关键创新点**（2-3个，列出解决了什么问题）
3. **损失函数**（设计合理的训练目标）
4. **训练策略**（两阶段/端到端/预训练微调）

### 图像超分辨率设计方案

**参考框架**：

> **总体思路**：深度学习超分主流是 SRCNN → SRResNet → ESRGAN → SwinIR。创新方向可以是"频率域+空间域双分支"或"扩散模型超分"。

**方案一（推荐，稳妥有亮点）：双分支频率-空间超分网络**

1. **结构**：
   - 浅层特征提取：一层卷积
   - **空间分支**：多个残差块（含注意力），提取空间纹理
   - **频率分支**：DCT/FFT 变换后在频率域处理，再逆变换回空间域
   - **融合模块**：可学习的加权融合 + 1×1 卷积
   - **上采样模块**：PixelShuffle（亚像素卷积）+ 残差连接

2. **创新点**：
   - 频率分支能更好地恢复高频细节（边缘、纹理）
   - 双分支自适应融合，对平滑区域和纹理区域分别处理

3. **损失函数**：$L = L_1 + \lambda_1 L_{perceptual} + \lambda_2 L_{GAN} + \lambda_3 L_{frequency}$
   - $L_{frequency}$：频率域的 L1 损失，约束高频分量

**方案二（追热点）：扩散模型超分**

1. **结构**：条件扩散模型，以低分图为条件逐步去噪生成高清图
2. **创新点**：在扩散过程中引入频率域约束，加速采样

---

### 神经机器翻译设计方案

**参考框架**：

> **总体思路**：NMT 主流是 Transformer + 预训练 + 微调。创新方向可以是"显式篇章级建模"或"多粒度融合"。

**方案：多粒度混合翻译模型**

1. **基础**：Transformer Encoder-Decoder

2. **创新点一：多粒度编码**
   - **词级**：标准 BPE/子词编码
   - **短语级**：用 CNN 或 n-gram 注意力捕获短语模式
   - **句级**：在编码器顶层做句级对比学习
   - 三种粒度通过门控机制融合

3. **创新点二：双向上下文增强**
   - 用两个编码器（正向 + 反向），通过交叉注意力融合
   - 解决单向语言模型只看左侧上下文的问题

4. **创新点三：自适应解码策略**
   - 根据不同句子的难度动态调整 beam size
   - 用一个小型的 complexity estimator 预测翻译难度

5. **损失函数**：$L = L_{NLL} + \alpha L_{contrastive} + \beta L_{length}$
   - $L_{contrastive}$：拉近语义相似句对的表示
   - $L_{length}$：约束输出长度比例

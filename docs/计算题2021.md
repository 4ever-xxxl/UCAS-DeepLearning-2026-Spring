这是一个非常经典的机器学习损失函数计算题。根据题目提供的提示 $\lg 2$ 和 $\lg 3$ 的值，本题中的对数计算均采用**以 10 为底的常用对数（$\lg$）**。

在开始计算之前，我们先根据提示求出各个样本在公式中会用到的对数值。
实际输出即为预测为正类的概率 $\hat{y}$。

* $lg(0.8) = \lg(\frac{8}{10}) = 3\lg2 - 1 \approx 3 \times 0.301 - 1 = -0.097$
* $lg(0.6) = \lg(\frac{6}{10}) = \lg2 + \lg3 - 1 \approx 0.301 + 0.477 - 1 = -0.222$
* $lg(0.9) = \lg(\frac{9}{10}) = 2\lg3 - 1 \approx 2 \times 0.477 - 1 = -0.046$

已知样本 $N = 5$，真实标签 $y = [0, 1, 0, 0, 1]^T$，预测概率 $\hat{y} = [0.2, 0.8, 0.4, 0.1, 0.9]^T$。

---

### (1) 计算模型的交叉熵损失 (Cross-Entropy Loss)

二分类的交叉熵损失函数公式为（计算均值）：


$$CE = -\frac{1}{N} \sum_{i=1}^{N} \left[ y_i \lg(\hat{y}_i) + (1 - y_i) \lg(1 - \hat{y}_i) \right]$$

我们逐个计算每个样本的交叉熵损失 $CE_i$：

1. **样本 1** ($y_1=0, \hat{y}_1=0.2$):
$CE_1 = -\lg(1 - 0.2) = -\lg(0.8) = 0.097$
2. **样本 2** ($y_2=1, \hat{y}_2=0.8$):
$CE_2 = -\lg(0.8) = 0.097$
3. **样本 3** ($y_3=0, \hat{y}_3=0.4$):
$CE_3 = -\lg(1 - 0.4) = -\lg(0.6) = 0.222$
4. **样本 4** ($y_4=0, \hat{y}_4=0.1$):
$CE_4 = -\lg(1 - 0.1) = -\lg(0.9) = 0.046$
5. **样本 5** ($y_5=1, \hat{y}_5=0.9$):
$CE_5 = -\lg(0.9) = 0.046$

**总交叉熵损失：**


$$CE_{total} = 0.097 + 0.097 + 0.222 + 0.046 + 0.046 = 0.508$$

**模型的平均交叉熵损失：**


$$CE = \frac{0.508}{5} = 0.1016$$

### (2) 计算模型的焦点损失 (Focal Loss)

标准 Focal Loss 针对类别不平衡和难易样本引入了权重因子 $\alpha$ 和调节因子 $\gamma$。
公式为（计算均值）：


$$FL = \frac{1}{N} \sum_{i=1}^{N} FL_i$$


其中单样本损失 $FL_i$ 定义为：
$$FL_i = \begin{cases}
-\alpha (1 - \hat{y}_i)^\gamma \lg(\hat{y}_i), & y_i = 1 \
-(1 - \alpha) \hat{y}_i^\gamma \lg(1 - \hat{y}_i), & y_i = 0
\end{cases}$$

已知 $\gamma = 2, \alpha = 0.4$，则对于负样本权重系数 $1 - \alpha = 0.6$。逐个计算每个样本的 Focal Loss：

1. **样本 1** ($y_1=0, \hat{y}_1=0.2$): 属于负类
$FL_1 = -0.6 \times (0.2)^2 \times \lg(1 - 0.2)$
$= -0.6 \times 0.04 \times (-0.097) = 0.024 \times 0.097 = 0.002328$
2. **样本 2** ($y_2=1, \hat{y}_2=0.8$): 属于正类
$FL_2 = -0.4 \times (1 - 0.8)^2 \times \lg(0.8)$
$= -0.4 \times 0.04 \times (-0.097) = 0.016 \times 0.097 = 0.001552$
3. **样本 3** ($y_3=0, \hat{y}_3=0.4$): 属于负类
$FL_3 = -0.6 \times (0.4)^2 \times \lg(1 - 0.4)$
$= -0.6 \times 0.16 \times (-0.222) = 0.096 \times 0.222 = 0.021312$
4. **样本 4** ($y_4=0, \hat{y}_4=0.1$): 属于负类
$FL_4 = -0.6 \times (0.1)^2 \times \lg(1 - 0.1)$
$= -0.6 \times 0.01 \times (-0.046) = 0.006 \times 0.046 = 0.000276$
5. **样本 5** ($y_5=1, \hat{y}_5=0.9$): 属于正类
$FL_5 = -0.4 \times (1 - 0.9)^2 \times \lg(0.9)$
$= -0.4 \times 0.01 \times (-0.046) = 0.004 \times 0.046 = 0.000184$

**总焦点损失：**


$$FL_{total} = 0.002328 + 0.001552 + 0.021312 + 0.000276 + 0.000184 = 0.025652$$

**模型的平均焦点损失：**


$$FL = \frac{0.025652}{5} = 0.0051304$$

**总结:**
(1) 模型的（平均）交叉熵损失为 **$0.1016$** （总损失为 $0.508$）。
(2) 模型的（平均）焦点损失为 **$0.0051304$** （总损失为 $0.025652$）。

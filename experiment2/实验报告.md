# 实验二 基于 ViT 的 CIFAR10 图像分类实验报告

## 一、实验目的

1. 学习使用深度学习框架实现和训练 ViT 模型，理解 ViT 中的 Attention 机制。
2. 掌握图像分类任务中数据读取、网络构建、模型训练、验证和测试的完整流程。
3. 在 CIFAR-10 数据集上训练并评估模型，使测试集准确率达到 80% 以上。

## 二、实验环境

- 操作系统：Linux
- Python：3.13.12
- 深度学习框架：PyTorch 2.11.0+cu130、torchvision 0.26.0+cu130
- 训练硬件：NVIDIA GeForce RTX 5070
- 数据集：CIFAR-10

## 三、实验原理

ViT 将图像划分为 patch 序列，并使用 Transformer Encoder 建模图像不同区域之间的全局关系。模型主要由图像特征嵌入模块、Transformer 编码器模块和 MLP 分类模块组成。

本实验实现的模型流程如下：

`ConvPatchEmbed -> cls token + position embedding -> Transformer Encoder Blocks -> LayerNorm -> Linear classifier`

每个 Transformer Encoder Block 的结构如下：

`LayerNorm -> Multi-Head Self-Attention -> residual -> LayerNorm -> MLP -> residual`

考虑到 CIFAR-10 原始图像尺寸为 32x32，直接放大到 224x224 会显著增加训练时间。本实验在保留 ViT 编码器和多头注意力机制的基础上，使用卷积 tokenizer 直接处理 32x32 输入，减少 token 数量并增加局部特征提取能力。

## 四、实验步骤

1. 下载并加载 CIFAR-10 数据集。
2. 将训练集按 9:1 划分为训练集和验证集，测试集保持独立。
3. 对训练集使用 RandomCrop、RandomHorizontalFlip、RandAugment、RandomErasing 等数据增强。
4. 构建基于卷积 tokenizer 的 compact ViT 分类模型。
5. 使用 AdamW 优化器、warmup + cosine 学习率调度、MixUp 和 label smoothing 进行训练。
6. 每轮训练后在验证集上评估，保存验证准确率最高的模型。
7. 加载最佳模型，在测试集上计算最终损失和准确率。
8. 生成训练曲线、混淆矩阵、预测样本和第一层卷积特征图。

## 五、关键实现说明

- 数据预处理：`Normalize(mean=(0.4914, 0.4822, 0.4465), std=(0.2023, 0.1994, 0.2010))`
- 数据增强：`RandomCrop(32, padding=4)`、`RandomHorizontalFlip`、`RandAugment`、`RandomErasing`
- 正则化策略：MixUp、label smoothing、dropout、stochastic depth
- 损失函数：支持硬标签和 MixUp 软标签的交叉熵
- 优化器：`AdamW(lr=8e-4, weight_decay=0.05)`
- 学习率调度：线性 warmup 后使用 cosine annealing
- 模型保存：保存验证集准确率最高的 `best_model.pt`
- 可视化输出：`training_curves.png`、`confusion_matrix.png`、`predictions.png`、`feature_maps.png`

## 七、结果分析

1. 模型在第 12 个 epoch 时验证准确率已达到 80.02%，第 20 个 epoch 达到最佳验证准确率 84.60%，说明改进后的结构和训练配置能够较快达到实验指标。
2. 与直接将 CIFAR-10 放大到 224x224 的 ViT 相比，32x32 原生输入显著减少 token 序列长度和注意力计算量，有利于缩短训练时间。
3. 卷积 tokenizer 提供了局部特征提取能力，缓解了纯 ViT 在小数据集上缺少局部归纳偏置、容易过拟合的问题。
4. MixUp、RandAugment、RandomErasing 和 label smoothing 提升了训练样本多样性。由于 MixUp 会混合图像和标签，训练准确率不再等同于普通硬标签训练准确率，模型效果应主要参考验证集和测试集准确率。

## 八、结论

本实验完成了基于 ViT 的 CIFAR-10 图像分类任务，实现了数据加载、模型构建、训练、验证、测试和结果可视化的完整流程。最终模型在测试集上取得 83.36% 的准确率，超过实验要求的 80% 指标。
